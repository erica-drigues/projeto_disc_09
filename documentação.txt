A criação deste programa teve como objetivo desenvolver uma calculadora de IMC (Índice de Massa Corporal) utilizando o conceito de Programação Orientada a Objetos (POO) e a biblioteca tkinter para criar uma interface gráfica. O processo de desenvolvimento foi dividido em duas etapas principais: a implementação da lógica do cálculo do IMC e a construção da interface gráfica para interação com o usuário.

Inicialmente, começamos com a implementação da lógica do IMC em uma classe chamada IMC. Essa classe foi projetada para encapsular os dados e as operações relacionadas ao cálculo do índice. Dois atributos principais foram definidos: peso, representando o peso da pessoa em quilogramas, e altura, correspondendo à altura em metros. Esses atributos foram declarados como privados, seguindo os princípios da POO, para proteger os dados e garantir que só pudessem ser acessados ou modificados por meio de métodos específicos.

Para permitir a manipulação dos atributos privados, foram implementados métodos do tipo getter e setter. O getter permite acessar os valores de peso e altura, enquanto o setter realiza validações para assegurar que os valores atribuídos a esses atributos sejam positivos, evitando possíveis erros durante o cálculo.

Além disso, a classe conta com dois métodos principais: calcular_imc e classificar_imc. O método calcular_imc realiza o cálculo do índice utilizando a fórmula padrão do IMC, que é o peso dividido pelo quadrado da altura. Já o método classificar_imc determina a classificação do IMC com base nos padrões da Organização Mundial da Saúde (OMS). Essas classificações incluem categorias como "Abaixo do peso", "Peso normal", "Sobrepeso" e diferentes graus de obesidade, permitindo ao usuário compreender melhor sua situação de saúde.

Após a criação da lógica do cálculo do IMC, passamos à implementação da interface gráfica utilizando a biblioteca tkinter. O objetivo era criar uma aplicação amigável para que o usuário pudesse inserir seus dados, calcular o IMC e visualizar a classificação. A interface foi desenvolvida com campos de entrada para o peso e a altura, um botão para acionar o cálculo e uma área para exibir os resultados. O botão foi configurado para chamar uma função que utiliza os valores fornecidos pelo usuário, cria um objeto da classe IMC, realiza os cálculos e exibe os resultados na interface. Também foram adicionados tratamentos de erro para garantir que entradas inválidas, como valores negativos ou letras, fossem devidamente tratadas, exibindo mensagens explicativas.

Por fim, o programa foi testado para garantir seu funcionamento correto e sua usabilidade. A interface gráfica foi ajustada para ser clara e simples, tornando o processo de cálculo do IMC intuitivo mesmo para usuários com pouca experiência em tecnologia. O resultado é uma ferramenta funcional, que combina conceitos de POO com a praticidade de uma interface gráfica, proporcionando uma experiência completa ao usuário.
